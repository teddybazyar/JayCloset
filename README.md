# JayCloset
## Website Link: jaycloset.etowndb.com

## Overview
JayCloset is Elizabethtown College's Career Closet! If students are in need of business casual clothing, whether for an interview or a meet and greet, this website is the place to go. Students can create an account and reserve clothing in the library catalog.

## Creating an Account
If you need to create an account, head to the login page and hit **"Register Here".** You will be prompted for your student ID, email, first name, last name, and be asked to create a password. A password must contain a certain amount of characters, 1 uppercase, 1 lowercase, 1 number, and 1 special character.

## After Account Creation
After an account is created, you can see your profile, cart, and eventually order historys, after an order is placed.
